Analytics Dashboard Setup

One-time setup to connect your Google Analytics data.

  1. Go to Google Cloud Console and create a project (or select existing).
  2. Go to APIs & Services → Library, search for Google Analytics Data API and Enable it.
  3. Go to APIs & Services → OAuth consent screen. Choose External, enter app name (e.g. "Datadripco Dash"), save. Then add your Google email as a Test User.
  4. Go to Credentials → Create Credentials → OAuth 2.0 Client ID. Type: Web application.
  5. Add Authorized JavaScript origins: https://datadripco.com (and http://localhost:1313 for local dev).
  6. Copy the Client ID (looks like 123…apps.googleusercontent.com).
  7. Go to Google Analytics → Admin → Property Details. Copy your numeric Property ID.
  8. Open this file (static/dashboard/index.html) and paste both values into the CONFIG at the bottom of the file.

After saving, rebuild your site and refresh this page.
